# GeoGebra Patcher (ggb_patcher) 

GeoGebra without the eye cancer that is a forced light mode. (At least I couldn't find a option to enable dark mode)

### Why

- I use GeoGebra sometimes, and my school wants us to use GeoGebra
- I do not want eye cancer (yes I am a certified light mode hater)

So I created this simple patcher that hacks together a cheap dark mode.

### How to use

Download the installer executable. Run it. Done. (Assuming you have GeoGebra Calculator Suite / Classic 6 / any of the standalone apps (why) installed)

### How to update

Just download and run a newer installer executable. It will overwrite the old one.

### How to uninstall

Why would you want to uninstall the very thing that saves you from getting eye cancer? 

(Just download the uninstaller executable and run it.)

### How to contribute

Download golang. Download upx and put it in your path. Clone this repository. Make changes. Run the build script. 

### Linux?

So far Windows only as I primarily use GeoGebra on windows.

I'm also not sure how GeoGebra works on linux (if available at all). If you want it, build it for yourself, see if anything needs to be fixed, and contribute to the repository.
